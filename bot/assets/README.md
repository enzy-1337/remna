# Картинки для экранов бота

Положите сюда файл **`section_header.png`** (квадрат или баннер для «Профиль», «Подписка» и др.).

Либо задайте в `.env`:

- `BOT_SECTION_PHOTO_PATH=/полный/путь/к/файлу.png`
- или `BOT_SECTION_PHOTO_URL=https://...` (прямая ссылка на изображение)

Если ни файл, ни URL не заданы — бот покажет только текст (без фото).
